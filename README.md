# Gabriela Bolos e Doces

Site de vendas de bolos e doces artesanais.

## Descrição

O projeto é uma aplicação web para uma loja de bolos e doces, oferecendo uma interface atraente e funcional para clientes fazerem pedidos online. Desenvolvido em HTML, CSS e JavaScript, utilizando o framework Bootstrap para um design responsivo e moderno.

## Funcionalidades

- **Catálogo de Produtos**: Apresentação de bolos, docinhos, cupcakes, kits prontos e opção "Monte seu Kit"
- **Sistema de Pedidos**: Clientes podem selecionar produtos, escolher tamanho, quantidade e informar dados de entrega
- **Login/Cadastro**: Sistema de autenticação com Firebase Authentication (e-mail/senha e Google)
- **Área do Cliente**: Histórico de pedidos e acompanhamento de status
- **Área Administrativa**: Gerenciamento de pedidos pela Gabriela (confirmar, rejeitar, marcar como entregue)
- **Integração com WhatsApp**: Envio automático do pedido via WhatsApp para a loja

## Páginas

- **index.html** - Página inicial com produtos
- **login.html** - Login de clientes
- **cadastro.html** - Cadastro de novos clientes
- **pedidos.html** - Sistema de pedidos
- **meus-pedidos.html** - Histórico de pedidos do cliente
- **admin-pedidos.html** - Gerenciamento de pedidos (para a Gabriela)
- **perfil.html** - Perfil do cliente

## Tecnologias Utilizadas

- **HTML5**: Estrutura básica do site
- **CSS3**: Estilização e layout responsivo
- **JavaScript**: Interatividade e manipulação do DOM
- **Bootstrap 5**: Framework para design responsivo e componentes prontos
- **Firebase**: Backend como serviço (Authentication e Firestore)
